# iot-manager-node-emulator

## Install
```
git clone https://github.com/alexsuslov/iot-manager-node-emulator.git
cd iot-manager-node-emulator
npm i
```

### Use
#### Demo
```
node demo.js

```

#### fillgauge
```
node fillgauge.js

```

#### display-value
```
node display-value.js

```

#### simple-btn
```
node simple-btn.js

```

#### anydata
```
node boiler.js

```
